Chapter 3 Technik

Hier gebe ich dir nützliche Tipps für alles Digitale.

3.1 Mac Tricks

3.1.1 Öffne die Developer-Tools von Google-Chrome

  • command + shift + i

3.1.2 Speichern

  • command + s

3.1.3 Speichern Unter

  • shift + option + command + s

3.1.4 Finder

  • command + leerschlag

3.1.5 Programm schliessen

  • command + q

3.1.6 Neuer Ordner

  • shift + command + n

3.1.7 Tilde Zeichen

Das ist nützlich für shortcuts im Terminal, wenn man das Working Directory festlegt:

  • option + n

3.1.8 Backslash

Useful for paths within the Directory:

  • option + shift + 7

3.1.9 Screenshot vom Bildschirm

  • command + shift + 3

3.1.10 Teil des Bildschirms Screen-Schoten

  • shift + command + 4

3.1.11 Verlauf löschen in Chrome Browser

  • shift + command + entfernen

3.1.12 Verlauf löschen Safari

  • command + alt + e

3.1.13 Bildschirm Aufnahme

Funktioniert nur, wenn QuickTime Player aktiv ist:

  • option + command + n

3.1.14 Lesezeichen in Google Chrome

  • command + d

3.1.15 Search Console Pop Up

  • option + command + j

3.1.16 Format Kopieren einer Zelle (in Numbers)

  • option + command + c

3.1.17 Format übertragen einer Zelle (in Numbers)

  • option + command + v

3.1.18 Switch between Applications on your computer

  • command + tab

3.1.19 Move Forward through Tabs

  • control + tab

3.1.20 Zahl als Exponent

Geht nur in Pages oder Tablett!:

  • shift + control + command + „+“-Zeichen WÄHREND man die Zahl mit der Maus markiert

Dann ist sie noch „normal“ und nicht tiefgestellt —> also so: zum Beispiel: 2.

3.1.21 Source-Code einer Webseite aufschalten

  • Option + CMD + u im Browser drücken

3.1.22 Interaktive Code-Ansicht für das Abchecken von Webseiten

fn + F12-Taste auf Touch Bar drücken

3.1.23 Approximate Symbol ≈

  • option + x

3.2 Terminal Commands

3.2.1 Clear in Terminal

  • option + k

3.2.2 Open the Finder with current Wd

Type in the following within your terminal:

  • open ~

3.2.3 Print the current Working Directory

Gebe im Terminal ein:

  • pwd
